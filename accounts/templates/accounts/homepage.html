The project

Welcome to the project what we have here is a django project that demonstrates extending the django user model to give us custom registration fields including file uploads.

We also tackled adding real world functionalities like login,password reset,sending email, CRUD through a task manager that extends beyond Django basic functionalies using a mysql database.

On the right side of the scrern we have displayed the details you submitted during your registration. On the navbar we have placed a navlink to send you diredctly to your personal task manager everything on the application should be fully functinal.
